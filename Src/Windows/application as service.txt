Application as service is a web application with minimum of server side codes but necessary that can't be achieved with HTML, CSS and Javascript but using the maximum of client side codes with Ajax asynchronous requests. The goal is to have server discharge and client charge to avoid unavailable service like encounter a web service. The display of data is manage by Javascript and all that can be. So application is optimized and perform well. The errors are more a client side problem and so easily rescheduled.
